🎓 Práctica 3 – CI/CD con GitHub Actions y Deploy Automático a Heroku

En esta práctica se implementa un flujo completo de Integración Continua (CI) y Despliegue Continuo (CD) utilizando GitHub Actions.
La aplicación se testea automáticamente en cada push y, si todo pasa correctamente, se despliega en Heroku sin intervención manual.

🚀 Objetivo de la práctica

Crear una API mínima con Express.

Implementar tests automáticos con Jest + Supertest.

Configurar un workflow de GitHub Actions para:

Instalar dependencias

Ejecutar tests

Desplegar automáticamente en Heroku

Utilizar secretos seguros (HEROKU_API_KEY).
